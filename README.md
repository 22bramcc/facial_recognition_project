# Facial Recognition Project

## Introduction

This project focuses on building a facial recognition dataset and applying preprocessing techniques. The dataset contains images of multiple individuals organized into separate folders. A Python script resizes images, normalizes pixel values, and saves processed images for future machine learning tasks.

## Installation Instructions

1. Install Python (version 3.x recommended)
2. Install required library:
   pip install opencv-python

## Usage

1. Place images in the dataset folder organized by person.
2. Run the preprocessing script:
   python preprocess.py
3. Processed images will be saved in the processed folder.

## Features

* Organized dataset by individual
* Image resizing to 224x224 resolution
* Pixel value normalization
* Automated preprocessing script
* Maintains dataset folder structure
* Ready for machine learning applications
